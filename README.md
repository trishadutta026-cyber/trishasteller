# 💰 Royalty Engine Smart Contract (Soroban)

## 📌 Project Description

The **Royalty Engine** is a smart contract built using Soroban on the Stellar blockchain. It provides a decentralized and transparent way to manage and calculate royalties for creators, ensuring they receive a fair share of revenue from transactions involving their digital assets.

This project is ideal for use in NFT marketplaces, digital content platforms, and creator economy applications.

---

## ⚙️ What it does

The smart contract enables:

* Creators to register their royalty percentage
* Storage of royalty data on-chain
* Retrieval of royalty details anytime
* Automatic calculation of royalty amount based on transaction value

This eliminates the need for manual royalty calculations and ensures trustless execution.

## 🚀 How It Works

### 1. Set Royalty

Register a creator's royalty percentage:

```
set_royalty(creator_address, percentage)
```

---

### 2. Get Royalty

Retrieve the stored royalty percentage:

```
get_royalty(creator_address)
```

---

### 3. Calculate Royalty

Calculate royalty from a transaction amount:

```
calculate_royalty(creator_address, amount)
```
